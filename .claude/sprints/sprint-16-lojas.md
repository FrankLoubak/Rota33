# Sprint 16 — Publicação nas lojas · modelo líder: Fable 5 (mobile) + guia ao usuário

## Pré-requisito
Só inicia após o Sprint 15 `PASS` e o backend de produção estável (as lojas testam a API real).

## Escopo
Preparar os **artefatos técnicos** (build assinado, metadados, ícones, screenshots, contas
de teste) para Apple App Store e Google Play Store. Abertura de conta / envio final são
ações do **usuário** (seção 12 do plano).

## Definition of Done
- [ ] Build de produção mobile assinado: Android `.aab` (Play App Signing) e iOS (certificados/profiles).
- [ ] Metadados/ícones/screenshots nos tamanhos exigidos por cada loja.
- [ ] Política de Privacidade pública (uso de localização em segundo plano declarado).
- [ ] Conta de teste funcional para os revisores.
- [ ] IAP configurado se aplicável (Apple IAP / Google Play Billing).
- [ ] Guia passo a passo (seção 12) revisado com o usuário; envio final feito pelo usuário.

## Observação
Requisitos/taxas das lojas mudam — confirmar nos links oficiais antes de iniciar.
