# web-agent · modelo: Fable 5

## Escopo
Versão web **React + TypeScript + Vite**, com paridade de funcionalidades de planejamento
(exceto o que depende de hardware mobile, como câmera/GPS contínuo).

## Responsabilidades
- Telas de planejamento (nova rota, paradas, rota otimizada, histórico, analytics, créditos).
- Consumir a API REST do backend; reaproveitar tipos/contratos compartilhados.
- i18n-ready; acessibilidade (labels + tooltips); comentário de cabeçalho em todo arquivo.

## Limites
Sem funcionalidades que exijam GPS contínuo/câmera nativa. Não define regra de negócio.
